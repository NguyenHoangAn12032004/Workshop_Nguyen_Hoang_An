# FCAJ Bootcamp Kick-off & Orientation

### Event Details
| | |
|---|---|
| **Event Name** | First Cloud AI Journey — Bootcamp Kick-off & Orientation |
| **Date** | 17/04/2026 |
| **Location** | Online (via Google Meet / Zoom) |
| **Organizer** | AWS Study Group Vietnam |
| **Role** | Participant / Intern |

### Event Description
The Bootcamp Kick-off was the opening session of the **First Cloud AI Journey (FCAJ)** internship program, organized by the AWS Study Group community. The event introduced the 12-week program structure, learning objectives, and deliverables expected from each intern.

### Key Takeaways
- **Program Overview:** Understood the 12-week roadmap — from AWS foundations (weeks 1–4) to project implementation (weeks 5–9) and final reporting (weeks 10–12).
- **AWS Account Setup:** Received AWS Academy credentials and configured the development environment.
- **Mentor Introduction:** Met the mentoring team led by Nguyễn Gia Hưng (hunggia@amazon.com) who would guide the technical aspects of the internship.
- **Team Formation:** Connected with fellow interns and formed project groups for collaborative work.
- **Deliverables Checklist:** Received the complete list of required deliverables: worklog, proposal, blog posts, workshop, event participation, and self-evaluation.

### What I Learned
This orientation gave me a clear roadmap for the entire internship. Understanding the expected deliverables from day one allowed me to plan my time effectively and avoid last-minute rushes.
