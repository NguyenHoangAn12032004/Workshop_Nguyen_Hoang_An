### Week 2 Achievements:

* Mastered Amazon S3 management:
    * Bucket Policies for access control
    * Versioning for object history
    * Lifecycle transitions for cost optimization
    * Cross-origin resource sharing (CORS)

* Understood the differences between relational and NoSQL databases on AWS:
    * Amazon RDS: managed relational database (MySQL, PostgreSQL, etc.)
    * Amazon DynamoDB: fully managed NoSQL with millisecond latency

* Configured AWS IAM with proper security:
    * Created Users, Groups, and Roles
    * Attached Policies following least-privilege principle
    * Understood the difference between identity-based and resource-based policies

* Successfully connected an EC2-hosted application to RDS and implemented S3 file storage for documents and images.

* Gained ability to combine multiple AWS services in an integrated architecture.
