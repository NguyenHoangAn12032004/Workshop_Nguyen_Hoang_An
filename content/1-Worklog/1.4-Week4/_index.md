### Week 4 Achievements:

* Understood the AWS DevOps toolkit:
    * CodeCommit for version control (Git-based)
    * CodeBuild for compilation and test automation
    * CodePipeline for end-to-end CI/CD orchestration

* Successfully set up a CI/CD pipeline:
    * Source → Build → Test → Deploy stages
    * Automatic triggers on code commits
    * Deployment to Lambda/Amplify environments

* Created CloudWatch monitoring infrastructure:
    * Custom dashboards for system performance visualization
    * Alarm notifications for CPU, memory, error rate thresholds
    * Log groups for centralized log management

* Learned to track and optimize cloud costs:
    * AWS Cost Explorer for usage analysis
    * Budget alerts to prevent unexpected charges
    * Reserved capacity vs on-demand pricing strategies

* **Milestone:** Successfully deployed a complete web application (frontend + backend) through an automated pipeline with high availability.
